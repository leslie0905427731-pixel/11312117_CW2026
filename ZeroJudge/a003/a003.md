# 题目
a003. 兩光法師占卜術
# 解題思路
使用 switch-case 可以省掉三个 if
# 解題編號
18141567
# 程式碼
```cpp=
#include <iostream>
int main() {
    int m, d, s;
    std::cin >> m >> d;
    s = (m * 2 + d) % 3;
    switch (s)
    {
    case 0:
        std::cout << "普通" << std::endl;
        break;
    case 1:
        std::cout << "吉" << std::endl;
        break;
    case 2:
        std::cout << "大吉" << std::endl;
        break;
    default:
        break;
    } 
}
```
