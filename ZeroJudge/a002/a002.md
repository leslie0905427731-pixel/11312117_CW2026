# 题目
a002. 簡易加法
# 解題思路
这段代码实现了一个**简易加法器**，主要逻辑如下：

1.  **输入**：使用 `std::cin` 从控制台读取两个整数 `i` 和 `n`。
2.  **计算与输出**：将这两个数相加（`i + n`），并通过 `std::cout` 输出结果，最后换行。
3.  **返回**：`return 0` 表示程序正常结束。

简而言之，这是一个读取两个数并输出它们之和的入门级 C++ 程序。
# 解題編號
18141490
# 程式碼
```cpp=
#include <iostream>
int main() {
    int i, n;
    std::cin >> i >> n;
    std::cout << i + n << std::endl;

    return 0;
}
```
